
# **Steep Learning**

## **Getting Started**

## **Project Setup**

- Just click on the repo names and click the fork button and clone it!

### Nginx Setup

- This step assumes that you've installed nginx in your computer.

### Start steep learning web

- Dev Mode

  ```sh
  $ npm install
  ```
  ```sh
  $ npm run dev
  ```
  
- Production Mode
  ```sh
  $ npm install
  ```
  ```sh
  $ npm run build
  ```
  ```sh
  $ npm start
  ```
